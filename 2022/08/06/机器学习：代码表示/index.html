机器学习：代码表示

2022-08-06

1、代码预训练

使用大量代码语料和算力训练 bert 等模型，得到一个可以代码迁移学习的模型

2、下游任务

2.1 代码表示学习下游任务：

  • 代码翻译（code translation）就是给一段代码，让模型用另一种编程语言实现，使用 CodeSearchNet 数据集
    • 评价指标：BLEU 分数、准确率（完全匹配）、CodeBLEU 分数
  • 代码摘要（code summarization）和代码翻译类似，不同点是代码摘要的目的是生成代码的注释，用 CodeSearchNet 数据集
  • 代码搜索（code search）就是计算代码和代码注释的向量相似度（这个任务还有更细的划分，还没看明白）
  • 克隆检测（code detection）分为二分类任务和检索任务：二分类就是给定两个代码的向量表示，然后用模型去判断它俩是不是有相同的作用；检索任务就是给定一段代码，然后让模型找到和这个代码有相同作用的代码，这俩任务分别用 BigCloneBench 和 POJ-104 两个数据集
  • 代码修复（code repair）的数据集有俩（Java 长度小于 50 的和 Java 长度大于 50 小于 100 的），使用 Bugs2Fix 数据集
    • 评价指标：BLEU 分数，精确度、CodeBLEU 分数
