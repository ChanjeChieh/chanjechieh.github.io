基础排序算法

2020-05-08 算法 0 评论 字数统计: 646(字) 阅读时长: 2(分)

逆序对：
在一个数组里面，对于下标 i<j，如果 A[i]>A[j]，则称 (i,j) 或 A[i]>A[j] 是一对 逆序对。

一、冒泡排序

  • 将数组中 N 个元素看作 N 个大小不同的泡泡，从第一个元素开始 “冒泡”：遇到前一个元素比后一个元素大的时就交换两个元素的位置，这样，每次冒泡就将最大的泡泡放在 “最后” ，重复 N 次。
  • 如果有一次 “冒泡” 没有发现需要交换的两个相邻元素，说明数组已经排序完成，可直接退出。

    最好情况：O (N)
    最坏情况：O (N^2)
    优点：简单、稳定、可以处理链表的排序

二、插入排序

  • 将 N 个元素的数组看作 N 张牌，想象这样的场景：开始的时候手里只有下标为 0 处的那张 “牌”，接下来每次遍历中，拿到 当前下标加一 处的 “牌”，然后在在手里已经有的牌中找到合适的位置插入，重复这个过程。
  • 当所有的 “牌” 都在手中时，数组排序完成。

    最好情况：O (N)
    最坏情况：O (N^2)
    优点：简单、稳定

这两个排序算法虽然简单，但是每次交换相邻元素只能抵消一对逆序对，影响到了效率。那，提高效率的一个点就是想办法使每次交换元素时能抵消多个 逆序对，所以就有了下面的希尔排序。
