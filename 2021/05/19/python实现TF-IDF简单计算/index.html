python实现TF-IDF简单计算

2021-05-19 python, 机器学习 0 评论 字数统计: 586(字) 阅读时长: 2(分)

TF-IDF 是一种加权技术，常用于机器学习中。

假设我们有一个语料库 corpus，语料库中有许多文档 doc，每个文档 doc 中由许多单词 token 组成，对于这样一个语料库，TF-IDF 可以评估语料库中某个单词 token 对于语料库中某个文档的重要性，某个单词 token 的重要性与其在所在文档 doc 中出现的次数成正比，与该 token 在语料库中其他文档中出现的次数成反比。

TF-IDF 的思想为 某个单词在该文档中出现的频率较高，在其他文档中出现的频率较低，则该词汇可以很好地对文档进行分类，即具有较好的分类能力，其由两部分组成，其中 TF 为词频，表示某个单词 token 在其所在文档中出现的频数（次数），其公式为：
$$
TF(Token) = \frac{count_{doc}^{token}}{count_{doc}^{}}
$$
另一部分 IDF 为 逆文档频率，用总文档数除以包含该某个单词的文档数来表示。
$$
IDF(Token) = \frac{count_{corpus}^{}}{count_{corpus}^{doc\ which\ has\ token }}
$$
